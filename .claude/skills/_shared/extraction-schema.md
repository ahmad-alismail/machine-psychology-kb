# Extraction Schema (per source)

The working checklist for Pass 1 of ingestion — what to pull out of a source before
it is split across wiki pages. This is an ingestion artifact, not an output contract;
the finished pages follow the templates in `CLAUDE.md`.

```
### Metadata
- Title / Authors / Year / Venue·URL
### Relevance Tier
- CORE | SOURCE-METHOD
### Relevance to Machine Psychology   (REQUIRED if SOURCE-METHOD; for CORE only if the source draws it)
- the bridge: how this could be repurposed to evaluate AI behavior (wiki interpretation)
### Theories invoked or usable
- name → key prediction(s) → what it explains/predicts
### Paradigms / experimental designs
- name → setup → target behavior → scoring
### Instruments / measures
- inventories, benchmarks, scenarios used
### Constructs / behaviors probed
- map to safety-concepts/ where applicable
### Claims  (claim · evidence type · strength · section)
### Validity considerations
- construct validity · reliability · contamination · reproducibility
### Open questions / research gaps surfaced
```
