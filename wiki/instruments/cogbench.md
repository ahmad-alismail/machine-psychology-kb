# CogBench

> [!info] In plain terms
> A ready-made battery of psychology-lab tasks packaged as an AI benchmark — "a large
> language model walks into a psychology lab." Its special trick is that test items are
> generated by procedure rather than copied from a fixed list, so a model can't have
> memorized the answers from its training data. Run a model through it and you get ten
> numbers describing *how* it thinks — how risky, how exploratory, how planful — each scaled
> against the average human.

## What It Measures

Behavioral/cognitive profiles of LLMs — a **computational phenotype** — using seven canonical
experiments from the learning-and-decision-making literature. It yields **ten behavioral metrics**
plus a performance metric per task, all **human-normalized** (0 = random agent, 1 = average human):

| # | Task | Behavioral metric(s) |
|---|------|----------------------|
| 1 | Probabilistic reasoning (Dasgupta 2020) | prior weighting β₁, likelihood weighting β₂ |
| 2 | Horizon task (Wilson 2014) | directed exploration, random exploration |
| 3 | Restless bandit (Ershadmanesh 2023) | meta-cognition (adjusted QSR) |
| 4 | Instrumental learning (Lefebvre 2017) | learning rate, optimism bias (α⁺ − α⁻) |
| 5 | Two-step task (Daw 2011) | model-basedness |
| 6 | Temporal discounting (Ruggeri 2022) | discounting score (0–19) |
| 7 | BART (Lejuez 2002) | risk-taking (average inflation attempts) |

## Administration

Administered purely as prompts via in-context learning (no fine-tuning), at temperature 0;
past behavior is concatenated into the prompt (prompt-chaining) to induce learning. Items are
**procedurally generated** each run (instead of a static dataset), except temporal discounting.
Open-source: github.com/juliancodaforno/CogBench. Applied to 35 LLMs in the original study.

## Validity Notes

- Procedural generation makes it "inherently less susceptible to data contamination issues" — the
  recommended antidote to the contamination problem that plagues canonical psychology items.
- **External validity for LLMs is not yet established** — the tasks are externally validated for
  humans only ([[questions/cognitive-task-external-validity-for-llms]]).
- Trial counts were reduced from the human protocols (context-size limits), which may weaken fitted
  parameters; behavioral metrics (not performance alone) are needed because equal performance can
  hide opposite behavior.
